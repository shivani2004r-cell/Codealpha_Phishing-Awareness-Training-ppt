# 🛡️ Phishing Awareness Training

A cybersecurity awareness presentation developed as part of the **CodeAlpha Cyber Security Internship**. This project educates users about phishing attacks, social engineering techniques, fake websites, and best practices to stay safe online.

---

## 📌 Project Overview

Phishing is one of the most common cyber threats used by attackers to steal sensitive information such as usernames, passwords, banking details, and personal data. This presentation aims to spread awareness by explaining how phishing attacks work, how to recognize them, and how to prevent becoming a victim.

---

## 🎯 Objectives

- Understand what phishing is and why it is dangerous.
- Learn different types of phishing attacks.
- Identify phishing emails and fake websites.
- Understand social engineering techniques used by attackers.
- Learn cybersecurity best practices.
- Study real-world phishing examples.
- Test knowledge through an interactive quiz.

## 📚 Topics Covered

### ✔ What is Phishing?
Learn how attackers trick users into revealing confidential information.

### ✔ Types of Phishing
- Email Phishing
- Spear Phishing
- Whaling
- Smishing
- Vishing
- Social Media Phishing

### ✔ Recognizing Phishing Emails
- Suspicious sender
- Urgent language
- Fake links
- Unexpected attachments
- Requests for passwords or OTPs

### ✔ Fake Websites
- Verify HTTPS
- Check domain spelling
- Look for trusted certificates
- Identify suspicious URLs

### ✔ Social Engineering
- Impersonation
- Pretexting
- Baiting
- Scareware
- Quid Pro Quo

### ✔ Prevention Tips
- Verify email senders
- Hover over links before clicking
- Enable Multi-Factor Authentication (MFA)
- Use strong passwords
- Keep software updated
- Report suspicious emails
